# Студия маникюра и педикюра Елены Булеевой

Одностраничный сайт студии маникюра и педикюра в центре Новосибирска
(ул. Ленина, 12, оф. 1207). Чистый HTML/CSS/JS, без сборщиков.

## Локальный просмотр

```bash
python -m http.server 5577
# открыть http://localhost:5577
```

## Структура

- `index.html` — вся страница (стили и скрипты внутри).
- `images/` — фотографии работ, студии и руководителя.

Публикуется через GitHub Pages.
